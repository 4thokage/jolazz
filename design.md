# Hop Portugal — Design System (locked)

Portable design system extracted from the Hop Portugal MVP. Genre: **modern-minimal** (Apple Maps × Airbnb). Theme: **custom Hop Portugal** — warm, premium, calm.

> Generated by Hallmark. Source of truth: `src/routes/layout.css` (`@theme`) and `tokens.css`.

## Principles

- Discover, don't scroll. One action per screen.
- Desaturated Deep Hop Green — never neon, never black backgrounds.
- Large serif display (Fraunces) + Inter body. Generous whitespace.
- Outline icons only, consistent 1.75 stroke, no beer-mug skeuomorphs.
- Motion: transform/opacity only, ≤200ms, Hallmark easings; respects reduced-motion.

## Typography

- Display: **Fraunces** 400/500/600, `font-style: normal`.
- Body: **Inter** 400/500/600.
- Scale: `--text-display` (clamp 2.25–3.25rem) → `--text-xs`.

## Radius / Spacing / Elevation

- Cards `1.25rem`, chips `999px`, sheets `1.75rem`.
- 4pt spacing scale (`--space-xs` … `--space-3xl`).
- Shadows: `--shadow-card`, `--shadow-sheet`, `--shadow-tab`.

## Components (8-state)

`Button` · `Chip` · `BreweryCard` · `BottomTabBar` · `BottomSheet` · `FilterSheet` · `Toggle` · `SearchBar` · `EmptyState` · `Skeleton` · `ErrorState` · `StampBadge` · `Icon`.

## Motion

`--ease-out` / `--ease-in` / `--ease-in-out`; `--dur-fast` 140ms, `--dur-base` 200ms. Reduced-motion collapses all to ~0ms.

## Navigation

Persistent bottom tab bar (Explore · Search · Map · Favorites). Settings reachable from Explore header. Passport is a tab within Favorites.
